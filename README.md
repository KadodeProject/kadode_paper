## ビルド

PlatformIO を用いる.

## 開発時の注意

- Windows の場合は WSL でなく、C ドライブ配下に clone する(M5Paper への書き込みができない)

書き込み用の PlatformIO を発動するために 同封の kadode_paper.code-workspace で起動する(階層が一つしたなのでそのままだと自動しない)
