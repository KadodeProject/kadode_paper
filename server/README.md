# スクリーンショット生成サーバー

しくみ

表示画面の生成は[かどでポータル](https://portal.kado.day)に移譲。(管理しやすさのため)

M5Paper 側で画面を描画する手間が厄介であるため、ここでかどでポータル内の M5Paper 表示用ページを定期的にスクリーンショットとして保存し、それを M5Paper で読み出す仕組み。  
力技だが現状最適解であると考えられる(かどでペーパー側から POST リクエストを送ることはないため)
